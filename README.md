# NovaMind — AI Chat Interface

  Профессиональный чат-интерфейс для работы с AI-моделями. Поддерживает Groq, OpenRouter и Google Gemini.

  ## Сайт
  https://ai.aliterra.space/

  ## Установка и разработка
  ```bash
  npm install -g pnpm
  pnpm install
  PORT=3000 BASE_PATH="/" pnpm run dev
  ```

  ## Сборка
  ```bash
  PORT=3000 BASE_PATH="/" pnpm run build
  ```

  ## Провайдеры (бесплатно)
  - **Groq**: https://console.groq.com
  - **OpenRouter**: https://openrouter.ai/keys  
  - **Google Gemini**: https://aistudio.google.com/apikey
